# LectorPDF
Software lector de pdf´s:

  - Tamaño del archivo.
  - Tamaño de página.
  - Número de páginas.
  - Título.
  - Asunto.
  - Palabras Clave.
  - Tipo de archivo PDF.
  - Versión de PDF.
  - Lista de imágenes que pueda tener el documento.
  - Lista de fuentes que pueda incluir el documento.
